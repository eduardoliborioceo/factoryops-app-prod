1. PCP — Planejamento e Controle da Produção

Gestão de Ordens de Produção, roteiros, apontamentos e dados coletados

1.1 Controle de OPs

Cadastro e gestão das Ordens de Produção ativas.

Como cadastrar uma OP

  1. Acesse PCP → Controle de OPs.
  2. Clique em Nova OP.
  3. Preencha: número da OP, modelo/produto, quantidade e setor.
  4. O sistema verifica automaticamente se o número de OP já existe.
  5. Confirme clicando em Salvar.

Campos importantes

  • Número da OP: identificador único — geralmente termina em dois dígitos que indicam o setor.
  • Modelo: código do produto (ex.: 240170).
  • Saldo: quantidade ainda disponível para apontamento.
  • Status: calculado automaticamente conforme os apontamentos.

Atenção — OPs em família

OPs de uma mesma produção compartilham a base do número (ex.: 12345-01 e 12345-02). O sistema valida a sequência do roteiro: não é possível apontar um setor posterior sem produção registrada no setor anterior.

1.2 Roteiros de Produção

Define a sequência de setores que cada modelo deve percorrer.

O que é um roteiro?

O roteiro lista os setores pelos quais um modelo passa na ordem correta — por exemplo: PTH → SMD → IM → PA. O sistema usa o roteiro para bloquear apontamentos fora de sequência.

Como cadastrar

  1. Acesse PCP → Roteiros.
  2. Selecione o modelo e adicione os setores na ordem correta.
  3. Salve — o roteiro passa a valer imediatamente para novos apontamentos.

1.3 Apontamento de Produção

Vincule a produção coletada a uma OP específica.

Como vincular uma produção a uma OP

  1. Acesse PCP → Apontamento.
  2. Use os filtros para encontrar a linha/turno/data desejada.
  3. Clique em Vincular na linha de produção.
  4. Selecione a OP correta na lista de OPs abertas.
  5. Informe a quantidade e, se for SMD, informe a fase (TOP ou BOTTOM).
  6. Confirme com Vincular.

Vincular a múltiplas OPs

Uma mesma produção pode ser vinculada a mais de uma OP. Após a primeira vinculação, o modal permanece aberto para que você adicione outra OP sem precisar reabrir. O saldo de cada OP é atualizado em tempo real.

Validações automáticas do sistema

  • Modelo compatível: a OP deve ser do mesmo modelo da produção.
  • Saldo suficiente: a quantidade não pode exceder o saldo da OP.
  • Sequência do roteiro: o setor anterior deve ter produção registrada antes.
  • Fase SMD: OPs SMD exigem a informação de TOP ou BOTTOM.

1.4 Planejamento (Plano de Voo)

Planejamento diário da produção por linha e turno.

Como criar um planejamento

  1. Acesse PCP → Planejamento.
  2. Selecione a data, turno, setor e linha.
  3. Informe o modelo e a meta de produção.
  4. Salve — o planejamento aparece no resumo do dia.

Resumo planejado vs. realizado

O sistema compara automaticamente o que foi planejado com o que foi produzido, exibindo a eficiência (%) e os gaps de produção por linha e turno.

1.5 Produção Coletada — Input BI

Visualização dos dados de produção importados automaticamente pelo coletor.

Filtros disponíveis

  • Data inicial / Data final: período desejado.
  • Setor: filtra por área produtiva (PTH, SMD, IM, PA…).
  • Linha: filtra por linha de produção.
  • Turno: 1º, 2º ou 3º turno.

Importação manual

Caso o coletor não tenha enviado os dados, é possível importar manualmente um arquivo .json exportado do BI. Use o botão Importar JSON e acompanhe o progresso em tempo real.

Origem dos dados

Os registros do Input BI têm origem NULL ou sem marcação especial. Registros com origem mes pertencem à aba de Produção MES e não aparecem aqui.

1.6 Produção MES

Dados de produção importados do sistema MES, com meta e eficiência.

O que aparece aqui?

Registros com origem mes — importados mensalmente via JSON do MES. Incluem campo de meta por hora, permitindo calcular a eficiência real (% meta x real).

Como importar

  1. Exporte o arquivo do MES no formato .json.
  2. Acesse PCP → Produção MES.
  3. Clique em Importar JSON e selecione o arquivo.
  4. Acompanhe o progresso — registros duplicados são atualizados automaticamente.

1.7 Lançamento de Produção — PTH / Manual

Registro manual de produção para linhas que não têm coleta automática.

Quando usar

Use o Lançamento Manual para setores como PTH, IM e PA quando não há coleta automática via coletor ou quando for necessário registrar um lote avulso que não consta no Input BI.

Como lançar

  1. Acesse PCP → Lançamento de Produção.
  2. Preencha: data, turno, setor, linha, modelo e quantidade produzida.
  3. Opcionalmente informe perdas, defeitos e observação.
  4. Salve — o lançamento fica disponível para apontamento imediatamente.

Validação de roteiro

O lançamento manual também é considerado na validação de sequência do roteiro. Se o setor anterior não tem produção no Input BI, o sistema verifica se existe um lançamento manual para liberar o apontamento no setor seguinte.

2. SMT — Surface Mount Technology

Gestão de modelos, horas coletadas e relatórios do setor SMT

Modelos Cadastrados

Cadastro dos modelos que passam pelo setor SMT com seus parâmetros de produção. Cada modelo pode ter configurações específicas de tempo de ciclo e fases (TOP/BOTTOM).

Horas Coletadas

Visualização das horas produtivas e paradas coletadas pelo sistema. Permite identificar gargalos e calcular a eficiência real das linhas SMT.

Relatório de Horas

Relatório consolidado das horas por turno, linha e período. Exportável para análise de OEE e eficiência do setor.

Monitor em Tempo Real

Painel que exibe o status atual das linhas SMT: modelo em produção, quantidade produzida, paradas ativas e eficiência do turno corrente.

3. SFC — Shop Floor Control

Monitoramento do chão de fábrica com coleta de dados em tempo real

Painel de Acompanhamento

Tela de monitoramento projetada para TVs no chão de fábrica. Exibe a produção por linha em tempo real, com indicadores visuais de status (produzindo, parada, setup).

Relatório de Horas SFC

Consolidado das horas coletadas pelo SFC, com separação por turno, linha e tipo de evento (produção, parada programada, parada não programada).

4. Logística — Entregas e Expedição

Controle de entregas, lotes e rastreabilidade

Entregas

Registro e acompanhamento das entregas programadas. Permite associar lotes a ordens de entrega e gerar o comprovante de expedição.

Rastreabilidade

Consulta de lotes e rastreamento do histórico de cada produto: por qual linha passou, em que turno e qual operador registrou.

5. Configurações do Sistema

Administração de turnos, linhas e paradas — requer perfil Administrador

Turnos

Define os horários de início e fim de cada turno por setor. Esses horários são usados para classificar os registros de produção no turno correto.

Linhas

Cadastro e manutenção das linhas de produção por setor. Toda linha usada em apontamentos deve estar cadastrada aqui.

Paradas

Configuração dos tipos de parada programada (intervalo, refeição, ginástica, DDS, setup). Também permite cadastrar responsáveis (líder) e headcount por linha e turno.

Acesso restrito

As telas de Turnos, Linhas e Paradas são acessíveis apenas por usuários com perfil Administrador. Caso não veja essas opções no menu, solicite acesso ao responsável do sistema.

6. Acesso e Perfil de Usuário

Login, senha e navegação no sistema

Login

O acesso é feito com usuário e senha cadastrados pelo administrador. A senha deve ter no mínimo 10 caracteres, com letras maiúsculas, minúsculas, números e caractere especial.

Troca de senha

O sistema exige troca de senha a cada 180 dias. Quando o prazo vencer, o sistema solicitará automaticamente a criação de uma nova senha antes de liberar o acesso.

Navegação no menu

  • No desktop: use o menu lateral à esquerda com grupos expansíveis.
  • No mobile: use a barra de navegação inferior com os atalhos principais.
  • O botão ativa o modo tela cheia (oculta sidebar).

Perfis de acesso

  • Usuário comum: acesso às telas operacionais (PCP, SMT, SFC, Logística).
  • Administrador: acesso completo, incluindo Configurações do Sistema e gestão de usuários.
